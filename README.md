# Babar Ali — Portfolio (React + JavaScript)

Premium dark futuristic portfolio built with **React.js**, **React Router DOM**, **Tailwind CSS**, **Framer Motion** and **GSAP**.

## 🚀 Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173

## 🏗️ Build

```bash
npm run build
npm run preview
```

- Glassmorphism • Neon Blue + Purple Gradients
- Fully Responsive • SEO Optimized • Back to Top
- React Router DOM • Vite (Fast Build)

## ☁️ Deploy on Vercel

1. Push this folder to GitHub.
2. Import the repo on https://vercel.com.
3. Framework: **Vite**, Build: `npm run build`, Output: `dist`.
4. Deploy. Done.
